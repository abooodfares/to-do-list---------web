# 📝 To-Do List Web App

This is a simple and clean **To-Do List** web application that allows users to manage daily tasks directly in their browser.

---

## 🔍 Features

- ✅ Add new tasks
- 🗑️ Delete individual tasks
- ✔️ Mark tasks as completed
- 💾 Saves tasks in **Local Storage**
- 💻 Responsive UI – Works on desktop and mobile
